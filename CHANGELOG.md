# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-12-02

### Added
- Initial release of ZUK - DuckDuckGo CLI
- Interactive TUI built with Bubble Tea
- DuckDuckGo Lite API integration
- Search results parsing with goquery
- Browser integration for opening links
- Cross-platform support (Linux, macOS, Windows)
- Vim-style keyboard navigation (j/k)
- Clean project structure with cmd/internal layout

### Features
- Privacy-focused search (no tracking)
- Fast and lightweight single binary
- Real-time search with loading states
- Result snippets display
- Default browser integration
